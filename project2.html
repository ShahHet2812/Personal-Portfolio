Project Overview

A simple Python project that allows users to monitor, trade, and manage stocks with live price updates using real-time market data.

Key Features

  • Real-time Stock Prices: Fetch live market data to display up-to-date stock prices.
  • Stock Viewing: View detailed information about individual stocks, including current price, volume, and historical performance.
  • Buy/Sell Functionality: Execute stock purchases and sales with real-time price tracking.
  • Data Visualization: Display charts and graphs to analyze stock trends and performance over time.

Technologies Used

Python yfinance numpy matplotlib
