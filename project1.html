Personal Health Tracker

A Java Based Application to track the Health,Book Appointments,Calculate BMI and Maintain Patient Record

Project Overview

A Java-based Personal Health Tracker system that allows users to monitor and manage their fitness, nutrition, and overall wellness through personalized data tracking and analytics.

Key Features

  • User Profile Management: Create and customize individual health profiles with personal details and goals.
  • Activity Tracking: Log daily physical activities like steps, workouts, and calories burned.
  • Health Metrics Dashboard: View real-time health metrics like weight, BMI, heart rate, and sleep patterns.
  • Reminders & Notifications: Receive reminders for workouts, meals, hydration, and medication.
